# Assignement 2 for Standford course cs224d Deep Learning for Natural Language Processing http://cs224d.stanford.edu

## Detailed description of the assignement is in assignment2/assignment2.pdf

## How to run the code:
### 1. Install and activate virtual environment:
```bash
cd cs224d/assignment2/assignment2_release
pip2 install virtualenv
virtualenv -p python2 .env
source .env/bin/activate # Activate the virtual environment
```
### 2. Install dependencies:
```bash
pip2 install -r requirements.txt
```
### Note: installing modules from requirements.txt is crucial since the tensoflow API was changing a lot at the time of the creation of the code.
